# Examples

Examples for running the mc_luigi workflow.

* example_experiments.py defines the experiment
* input_config_train.json defines the configuration of the training data
* input_config_test.json defines the configuration of the test data

Use your own data or download example data from:
https://drive.google.com/open?id=0B4_sYa95eLJ1ek8yMWozTzhBbGM

Note that due to simplicity, we don't use a train/test split in this example.

For now, this only works for anisotropic 3d data with flat (2d) superpixels.
An isotropic version is on the way.
